Clearer ways to manipulate objects

/* Add an element to an array. */ var array = []; ~array.push(1, 0); ~array.unshift(2); array // returns [2, 1, 0] /* Add an element to an array. */ let array = [] ~.addElementToFront(1, 0) ~.addElementToBack(2); array // returns [2, 1, 0]

Strings are less complicated

/* Modify text like a pro. */ var str = 'Hello, World!'; ~// returns 'Hello' ~str.slice(0, str.indexOf(',')); ~// returns ' World!' ~str.slice(str.indexOf(',') + 1) /* Modify text like a pro. */ let str = 'Hello, World!'; ~// returns 'Hello' ~str.getBeforeChar(','); ~// returns ' World!' ~str.getAfterChar(',')

The developers of LapysJS believe that using core technologies creates a more seamless operational basis with the browser.
And while using Vanilla tech. may be missing some benefits, that is not to say that the library can not be modified and tweaked to improve upon it.

You can trust that with LapysJS, projects developed with the library will tend to be different from the norm using creative innovative styles & designs or
inventing better functionality & improving day-to-day web programming conventions.
LapysJS tries to strive for new, innovative and creative ideas.

LapysJS uses the current web APIs granted to developers with major polyfills in areas that could be lacking which overall make processing seamless with the browser.
