LapysJS

What is LapysJS?

LapysJS is a comprehensive, creative front-end web library that uses pure and raw HTML, CSS & JavaScript to build things otherwise difficult and stressful in the aforementioned core languages.
For most developers, using LapysJS cuts down large and tasking projects into smaller, more identifiable tasks.

Features

Vanilla is Better

With everything coded fresh and pure in LapysJS, the library is completely seamless with any project you work on while being able to handle all of the processing it does backstage.
This means no lagging, no buggy components & more speed.

Creative, New & Experimental

LapysJS is a constantly developing library that tries to be the outlier in the hook of everyday web development frameworks and other libraries.
It does this by developing for projects meant to stand out or
building creative, new components used only by the budding few.

Interoperable

LapysJS uses the current web APIs granted to developers with major polyfills in areas that could be lacking which overall make processing seamless with the browser.
