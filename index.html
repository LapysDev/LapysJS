Built by & for Minimalists

Took 2+ years or so. Was determined not to make it another JavaScript framework or jQuery clone (even though that was what it is was).

Struggled with the concept for a while until I figured out the library was designed for simplicity by maximizing native JavaScript features effectively rather than piling onto the stack with more custom features (like templates in React Native).

Anyways, it’s been fun developing this. Of course, this library was not built for profit but honestly for fun and the developer community.
Mostly to send a message that through critical & logical thinking, anything is possible even with 0’s and 1’s (the most minimal and simply sophisticated you can get).

Enjoy.
